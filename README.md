# 🏠 Ijaraga Uylar Bot

## O'rnatish yo'riqnomasi

### 1. Yopiq kanal invite linkini oling
1. Yopiq kanalingizga kiring
2. Kanal sozlamalari → Invite links → Create new link
3. `bot.py` faylida `YOPIQ_KANAL_LINK` ga o'sha linkni yozing

### 2. Botni yopiq kanalga admin qiling
1. Yopiq kanalga → Administrators → Add admin
2. `@Ijaraga_uylarUzbot` ni qidiring
3. Qo'shing (faqat "Read Messages" kerak)

### 3. Railway.app ga deploy qilish
1. https://railway.app ga kiring (GitHub bilan)
2. "New Project" → "Deploy from GitHub repo"
3. Bu fayllarni GitHub ga yuklang
4. Deploy tugmasini bosing

### 4. Ishlatish
- `/elon` - test post ko'rish
- `/kanalpost` - kanalga post yuborish (admin uchun)
